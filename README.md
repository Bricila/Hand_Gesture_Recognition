# Hand_Gesture_Recognition
## Description
This project detects hand landmarks using MediaPipe and classifies hand gestures
in real time using a KNN machine learning model.

## How to Run
pip install -r requirements.txt  
python scripts/predict_realtime.py

## Dataset
The dataset was collected locally using MediaPipe hand landmarks.

## Supported Gestures
The model is trained to recognize the following hand gestures:
- Peace
- OK
- Call
- Good Luck
- I Love You
- Loser
